ACL1
    -name: 101
    - Bloquear solo trafico FTP a Enterprise Web Server desde internet
    - Permitir el acceso al trafico
        Router# config t
        Router(config)# access-list 101 permit tcp any host <dirección_IP_servidor_web> eq ftp
        Router(config)# access-list 101 deny ip any any

ACL2
    - name: 111
    - Ningun host de HQ LAN 1 acceder al servidor de sucursal
    - Todo el trafico es permitido
        Router(config)#access-list 111 deny ip <dirección_IP_rango_HQ_LAN1> <máscara_de_subred_HQ_LAN1> <dirección_IP_servidor_sucursal> <máscara_de_subred_servidor_sucursal>
        Router(config)#access-list 111 permit ip any any
